# AmbientPedal 🎸

A JUCE-based real-time guitar effects app built in C++.

## Features
- Real-time audio input/output
- Delay effect
- Toggle ON/OFF
- Device selector

## Setup

1. Install JUCE
2. Open AmbientPedal.jucer in Projucer
3. Set module path to JUCE/modules
4. Save project
5. Build:

cd Builds/LinuxMakefile
make

## Run

./build/AmbientPedal
